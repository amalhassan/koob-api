# Koob-api Capstone Project

## Description

Koob.io brings together the latest news from around the world for your convenience. The articles come from multiple sources, such as New York Post, BBC News, Mashable, The Guardian, TechCrunch and more! The app gives you the space to enjoy your daily reads, morning or evening. ### Doppler secret link for environment variables

### `npm run server`

Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
